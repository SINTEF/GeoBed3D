GeoJSON Styling Guide

GeoBed3D reads standard GeoJSON files and uses a set of custom properties to control how each feature looks on the 3D globe. Add these properties inside the properties object of each feature. Properties not listed here are displayed in the info panel when you click a feature.

Shared properties — work on all geometry types

PropertyTypeDescription
color hex string Feature colour — #ff0000, #3b82f6, etc. Overrides the layer auto-colour.
title string Label shown floating above the feature on the map. Also used as the heading in the info panel.
note string / HTML Description shown in the info panel when the feature is clicked. Supports a limited set of HTML tags:

<strong></strong> — bold text
<i></i> — italic text
<br> — line break
<a href="…"></a> — hyperlink (always opens in a new tab)

Example: "Visit <a href='https://example.com'>the site</a> for more."
noteimage URL string Image URL shown in the info panel below the title. Any publicly accessible image works.

Points Point features

A point is rendered as a teardrop pin by default. Supply an icon URL to replace it with any image.

PropertyValuesDescription
iconsize small · medium · large Controls the size of the pin or icon. Default: medium. Also scales the title label offset.
icon URL string Replaces the default teardrop pin with a custom image. PNG with transparency works best. The image is loaded via the local proxy so CORS is not a concern.
color hex string Tints the default teardrop pin. Has no effect when a custom icon is set.
title string Text label rendered inside/above the pin on the map.

Icon sizes (pixels)

iconsizePin sizeDot fallback size
small24 × 24 px7 px radius
medium36 × 36 px10 px radius
large56 × 56 px16 px radius

Example

{
  "type": "Feature",
  "geometry": {
    "type": "Point",
    "coordinates": [23.7275, 37.9838]
  },
  "properties": {
    "title":     "Acropolis",
    "note":      "Ancient citadel of Athens",
    "noteimage": "https://example.com/acropolis.jpg",
    "color":     "#f59e0b",
    "iconsize":  "large"
  }
}

Example with custom icon

{
  "type": "Feature",
  "geometry": { "type": "Point", "coordinates": [23.7275, 37.9838] },
  "properties": {
    "title":    "Museum",
    "icon":     "https://example.com/museum-icon.png",
    "iconsize": "medium"
  }
}

Polygons Polygon features

Polygons are draped on the terrain surface and filled with a semi-transparent colour.

PropertyValuesDescription
color hex string Stroke and fill colour. The fill is automatically made semi-transparent based on opacity.
opacity 0 – 100 Fill opacity as a percentage. 0 = fully transparent, 100 = fully opaque. Default: 75.
title string Label rendered at the polygon centroid on the map.
Tip: Keep opacity between 30–60 for best results — the terrain texture shows through and the polygon remains clearly visible.

Example

{
  "type": "Feature",
  "geometry": {
    "type": "Polygon",
    "coordinates": [[
      [23.71, 37.97], [23.74, 37.97],
      [23.74, 37.94], [23.71, 37.94],
      [23.71, 37.97]
    ]]
  },
  "properties": {
    "title":   "Historic Centre",
    "color":   "#3b82f6",
    "opacity": 40
  }
}

Lines LineString features

Lines are rendered as solid-colour polylines clamped to the terrain surface at a fixed width of 10 px.

PropertyValuesDescription
color hex string Solid line colour. If omitted the layer auto-colour is used.
title string Label shown at the first coordinate of the line.

Example

{
  "type": "Feature",
  "geometry": {
    "type": "LineString",
    "coordinates": [
      [23.71, 37.98],
      [23.73, 37.96],
      [23.75, 37.95]
    ]
  },
  "properties": {
    "title": "Walking route",
    "color": "#10b981"
  }
}

Full FeatureCollection example

{
  "type": "FeatureCollection",
  "features": [

    // Point — large amber pin with label and info panel content
    {
      "type": "Feature",
      "geometry": { "type": "Point", "coordinates": [23.7275, 37.9838] },
      "properties": {
        "title":     "Acropolis",
        "note":      "Ancient citadel built in the 5th century BC",
        "noteimage": "https://upload.wikimedia.org/wikipedia/commons/thumb/d/da/The_Parthenon_in_Athens.jpg/320px-The_Parthenon_in_Athens.jpg",
        "color":     "#f59e0b",
        "iconsize":  "large"
      }
    },

    // Polygon — blue zone at 40% opacity
    {
      "type": "Feature",
      "geometry": {
        "type": "Polygon",
        "coordinates": [[[23.71,37.97],[23.74,37.97],[23.74,37.94],[23.71,37.94],[23.71,37.97]]]
      },
      "properties": {
        "title":   "Old Town",
        "color":   "#3b82f6",
        "opacity": 40
      }
    },

    // Line — green route
    {
      "type": "Feature",
      "geometry": {
        "type": "LineString",
        "coordinates": [[23.71,37.98],[23.73,37.96],[23.75,37.95]]
      },
      "properties": {
        "title": "Walking route",
        "color": "#10b981"
      }
    }

  ]
}
