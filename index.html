GeoJSON format reference 
{
  "type": "FeatureCollection",

  "edges": [
    { "from": "co2",
      "to":   "sea_temp",
      "weight": 0.7 }
  ],

  "features": [{
    "type": "Feature",
    "geometry": {
      "type": "Point",
      "coordinates": [15.5, 69.3]
    },
    "properties": {
      "id":        "co2",
      "label":     "CO₂",
      "fcm_value": 0.5
    }
  }]
}
coordinates [lon, lat] — standard GeoJSON order
edges top-level array, not inside features
weight −1 to +1 · blue = positive, red = negative
fcm_value initial state 0–1 · defaults to 0.5
label optional · falls back to name or id
